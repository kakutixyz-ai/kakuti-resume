# KakutiResume - モダンな履歴書エディタ

KakutiResumeは、ブラウザベースでバックエンド不要の履歴書エディタです。**Notion風の編集体験**と**Typstの強力な組版能力**を組み合わせ、リアルタイムプレビュー、高性能なPDF生成、そしてモダンなテンプレートを提供します。

**🌐 オンラインで試す: [https://resume.kakuti.site/](https://resume.kakuti.site/)**

## ✨ 主な機能

- **🚀 ブラウザ内 Typst コンパイル (WASM)**: すべてのPDF生成ロジックがブラウザ内で動作します。サーバー不要で、高いプライバシー保護と高速なレスポンスを実現しています。
- **📝 Notion風ブロックエディタ**: テキスト、リスト、見出しなどの多様なコンテンツブロックをサポートし、直感的でスムーズな編集が可能です。
- **📋 多彩なテンプレート**:
  - **日本標準履歴書** ＆ **職務経歴書**。
  - **モダンな英文レジュメ (Western Style)**。
- **💾 ローカル保存**: 編集内容はブラウザのローカルストレージ（LocalStorage/IndexedDB）にリアルタイムで保存されます。アカウント登録不要で、データの紛失も防げます。
- **📂 Markdown 対応**: Markdown形式でのインポート・エクスポートに対応。データの移行やバックアップも簡単です。
- **📱 レスポンシブプレビュー**: TypstによってレンダリングされたPDF/SVGをリアルタイムで確認でき、プレビュー通りの出力が得られます。

## 🛠️ 技術スタック

- **フレームワーク**: [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **ビルドツール**: [Vite](https://vitejs.dev/)
- **組版エンジン**: [Typst](https://typst.app/) (WASM経由でブラウザ内で動作)
- **スタイリング**: [Tailwind CSS](https://tailwindcss.com/)
- **状態管理**: React Hooks & Context API

## 🚀 クイックスタート

ローカル環境で実行する場合：

1. **依存関係のインストール**:
   ```bash
   npm install
   ```

2. **開発サーバーの起動**:
   ```bash
   npm run dev
   ```

3. **アクセス**: ブラウザで `http://localhost:5173` を開きます。

## 📦 デプロイ

Vercelに最適化されています。GitHubにプッシュし、Vercelプロジェクトと連携するだけで自動的にデプロイされます。ルートディレクトリの `vercel.json` にビルド設定が含まれています。

---

> **アーキテクチャの変更について**: 本プロジェクトは、サーバーサイドコンパイルからフロントエンドWASMコンパイルへと移行しました。以前の `backend` および `typst-service` ディレクトリは削除され、現在は純粋なシングルページアプリケーション (SPA) となっています。
